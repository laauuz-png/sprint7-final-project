El objetivo de la empresa es identificar patrones de uso, detectar comportamientos atípicos y comprender qué segmentos de clientes muestran necesidades diferenciadas, con el fin de optimizar la oferta comercial y mejorar la experiencia del usuario.
Utilicé 3 datasets: plans.csv: Catálogo de planes con sus precios y beneficios. users_latam.csv: Información de cada usuario (datos personales, plan, fecha de registro, churn) y usage.csv: Actividad generada por los usuarios: llamadas, mensajes, duración, longitud.
El proceso fue: cargar y explirar datasets, identificar nulos, sentinels y problemas de calidad, realizar limpieza básoca de datos, revisar estadísticas en variables numéricas y categóricas, realizar visualizaciones con histogramas y boxplots para identificar outliers, segmentar clientes y realizar un insight ejecutivo
